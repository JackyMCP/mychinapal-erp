Used here (embedded in generated quote PDFs) because it supports Polish
diacritics (ą ć ę ł ń ó ś ź ż) which jsPDF's built-in "helvetica" font does
not — those characters were rendering as missing glyphs / mis-spaced text.
Liberation Sans is metrically compatible with Arial/Helvetica, so the layout
built around "helvetica" measurements carries over cleanly.
